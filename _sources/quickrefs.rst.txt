|:ocean:| ML Flow
+++++++++++++++++++

Starting the ML Flow server

.. code-block:: bash 

    mlflow server --host 127.0.0.1 --port 6969


Purging an ML Flow experiment (run in the same directory where you run the server):

.. code-block:: bash

    mlflow gc --backend-store-uri ./mlruns
